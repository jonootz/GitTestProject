Branche

  • Multiple branches can be entirely independent of each other.
  • Uses:
    • Try out an ideas, merge back in
    • Experiment in, then delete branch
    • Traditionally used to test New Features
    • Have one that goes to production, another that you merge work into for testing, and several smaller ones for day to day work

Note: With Git you can switch between branches without having to commit. With Flare we force you to.
