Things to check in Flare:

You must commit all modified files to proceed with the synchronize:

  • Dirty file with no change on Remote - Syncs without saving or committing
  • Modified (Saved) file with no change on Remote - Forces Commit only
  • Committed file with no change on Remote - Nothing happens
  • Dirty file with change on Remote -Forces Commit only
  • Modified (Saved) file with change on Remote -Forces Commit only
  • Committed file with change on Remote -Nothing happens
  • After resolving conflicts - Yes, it Syncs
