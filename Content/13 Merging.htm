Merging

Currently no way to merge

Things to check in Flare:

Cannot merge branches in Flare

Things to check in Remote:

Can merge branches in Git via cmd line, then continue to work in Flare afterwards

 

Things to check in database via Git:

 

git merge <branch_name_of_one_to_merge_from>

First checkout (switch) to the branch you want to merge into, then use code above.

Any conflicts will also need to be resolved in Windows via a Notepad

 

git status:

Will show status of files in current, "CheckedOut" branch

 

git log:

Will log of commits in current, "CheckedOut" branch
